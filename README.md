# Orca Minecraft Mods

A collection of Minecraft Fabric mods for 1.21.1.

## Mods

| Mod | Description |
|-----|-------------|
| [Spinning Wheel](https://github.com/Simplifine-gamedev/spinwheel-mod) | Customizable spinning wheel for games and random decisions |
| [Balloon Pop](https://github.com/Simplifine-gamedev/balloon-mod) | Shootable balloons that drop loot when popped |
| [Aurora](https://github.com/Simplifine-gamedev/aurora-mod) | Aurora borealis visual effects |
| [Avalanche](https://github.com/Simplifine-gamedev/avalanche-mod) | Avalanche mechanics and effects |
| [Black Hole](https://github.com/Simplifine-gamedev/blackhole-mod) | Black hole entity with gravity effects |
| [Colossus](https://github.com/Simplifine-gamedev/colossus-mod) | Giant mob encounters |

## Requirements

All mods require:
- Minecraft 1.21.1
- Fabric Loader 0.16.0+
- Fabric API
- Java 21+
